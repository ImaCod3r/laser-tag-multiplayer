# laser-tag-multiplayer

Um jogo multiplayer de laser tag em tempo real com sistema de obstáculos (paredes).

## Características

- ✅ Multiplayer em tempo real via WebSockets (Socket.IO)
- ✅ Sistema de colisão com paredes
- ✅ Paredes como obstáculos estratégicos
- ✅ Sistema de saúde e dano
- ✅ Sistema de kills/pontuação
- ✅ Respawn automático
- ✅ Física de movimento e colisão circular

## Paredes/Obstáculos

As paredes são objetos retangulares que servem como obstáculos no mapa:
- Bloqueiam movimento dos jogadores
- Bloqueiam lasers
- Posicionadas estrategicamente no mapa para criar gameplay desafiador
- Colisão foi otimizada com detecção circular-retangular
